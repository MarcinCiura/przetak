# `przetak-sys`

Unsafe Rust bindings to the Przetak C library.

This crate provides low-level FFI bindings generated by [`bindgen`](https://github.com/rust-lang/rust-bindgen) from the Przetak shared library. It is not intended to be used directly; use the [`przetak`](../przetak/) crate for a safe interface.
